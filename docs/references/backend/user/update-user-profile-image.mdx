# `updateUserProfileImage()`

Updates a user's profile image.

```tsx
function updateUserProfileImage: (userId: string, params: { file: Blob | File }) => Promise<User>;
```

## Example

The following example shows two files:
1. A Next.js [Server Action](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations) that updates the user's profile image.
1. A Next.js [Client Component](https://nextjs.org/docs/app/building-your-application/rendering/client-components) that allows the user to upload the file.

<CodeBlockTabs options={["Server Action", "Client Component"]}>
```tsx {{ filename: 'app/actions/update-profile-image.ts', mark: [[19, 22]] }}
'use server';

import { clerkClient, auth } from '@clerk/nextjs/server';

export async function updateUserProfileImage(formData: FormData) {
  const { userId } = auth();

  if (!userId) {
    return Response.json(
      { message: 'No user found. Please sign in.' },
      { status: 401 }
    );
  }

  const params = {
    file: formData.get('file') as File,
  };

  const response = await clerkClient.users.updateUserProfileImage(
    userId,
    params
  );

  console.log(response);
}
```

## Backend API (BAPI) endpoint

This method in the SDK is a wrapper around the BAPI endpoint `POST/users/{user_id}/profile_image`. See the [BAPI reference](https://clerk.com/docs/reference/backend-api/tag/Users#operation/SetUserProfileImage) for more details.
